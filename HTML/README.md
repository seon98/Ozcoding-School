# Admin Page - 신규 등록 상품 관리 시스템

신규 등록 상품을 관리할 수 있는 간단한 Admin 페이지입니다.  
부트스트랩(Bootstrap)과 순수 자바스크립트로 구현했으며,  
상품 목록 조회, 카테고리 필터링, 검색, 정렬, 페이징, 다크 모드 토글 기능을 제공합니다.

---

## 주요 기능
- 상품 카테고리별 필터링 및 검색  
- 테이블 컬럼별 오름차순/내림차순 정렬 기능  
- 페이지네이션(5개씩 상품 목록 페이징)  
- 실시간 시계 표시  
- 다크 모드 / 라이트 모드 토글  
- 회원가입 폼 토글 표시

---

## 기술 스택
- HTML5, CSS3  
- Bootstrap 5  
- JavaScript (ES6)

---

## 사용법

1. 프로젝트를 클론 또는 다운로드 받습니다.
2. `index.html` 파일을 브라우저로 열면 바로 확인할 수 있습니다.
3. 검색창에 제품명을 입력하거나 카테고리를 선택 후 조회 버튼을 클릭하세요.
4. 테이블 헤더의 화살표 버튼을 클릭하여 정렬할 수 있습니다.
5. 페이지 하단의 페이징 버튼을 이용해 페이지를 이동하세요.
6. 오른쪽 상단의 "다크 모드" 버튼으로 테마를 변경할 수 있습니다.
7. "회원가입" 버튼을 클릭하면 간단한 회원가입 폼이 나타납니다.
